Este proyecto usa las siguientes tecnologías:
Front:

  • AngularJS
  • Bower
  • Gulp
  • Yeoman
Back:
  • Python 3.4
  • Django 1.8
  • Django Rest Framework
Server:
  • Vagrant
  • AWS: Elastic Beanstalk
Metodologías/Filosofías de trabajo:
  • TDD
  • SCRUM
  • Microservices

Al utilizar un repertorio tan grande de tecnologías, es un buen proyecto de referencia para crear proyectos distribuidos, con testing, etc...
Por supuesto, el código lo tenéis disponible en mi github para que podáis trastear todo lo que queráis. Espero vuestros pull requests ;)
